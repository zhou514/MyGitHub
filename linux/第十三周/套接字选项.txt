一. getsockopt
 相关函数
 setsockopt
 表头文件
 #include <sys/types.h>
 #include <sys/socket.h>
 定义函数
 int getsockopt(int s,int level,int optname,void *optval,socklen_t *optlen);
 函数说明
 取得socket状态，getsockopt()函数将参数s指定的socket状态返回。参数optname代表想要取得何种选项状态，参数optval指向想要保存结果的内存地址；
 参数optlen，内存地址大小；参数level，optname  参考setsockopt函数
 成功返回0 失败返回-1

 二 setsockopt
 相关函数
 getsockopt
 表头文件
 #include <sys/types.h>
 #include <sys/socket.h>
 定义函数
  int setsockopt(int s,int level,int optname,void *optval,,socklen_t *toplen);
  设置socket状态，setsockopt()设置参数s所指定的socket状态，level表示想要设置的网络层，一般设置为SOL_SOCKET来存储socket层；
  参数optname代表设置的选项：SO_DEBUG 打开或关闭排错模式； SO_REUSERADDR 允许在bind()过程中本地地址可以重复使用; SO_TYPE 返回socket形态 ; SO_ERROR 返回socket已发生的错误原因 ; SO_DONTROUTE 送出的数据包不要利用路由设备来传输 ； SO_BROADCAST 使用广播方式传送  ; SO_SNDBUF 设置送出的暂存区大小 ; SO_RCVBUF 设置接收的暂存区大小 ; SO_KEEPALIVE 定期检查确定连线是否终止 ; SO_OOBINLINE 当接收到OOB数据时马上送至标准输入设备 ; SO_LINGER 确保数据安全且可靠的传送出去；
  参数optval 代表想要设置的值  参数optlen 为optval的长度
  返回值  成功 返回0；失败返回-1
